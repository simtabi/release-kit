# Recipes

Copy-pasteable config snippets for common release shapes. Drop the
`release.json` body into your repo and edit the slugs.

| File | Use when |
|---|---|
| [`python-package-to-pypi.md`](python-package-to-pypi.md) | Pure Python package, PyPI + GitHub Releases. |
| [`node-package-to-npm.md`](node-package-to-npm.md) | TypeScript package, npm + GHCR + GitHub. |
| [`container-only.md`](container-only.md) | Docker images to GHCR + Docker Hub + ECR. |
| [`multi-registry-monorepo.md`](multi-registry-monorepo.md) | Mono-repo publishing distinct artifacts. |
